# 🛒 RetailIQ

A full-stack Inventory & Retail Management System built using Spring Boot, React, MySQL, Docker and Power BI.

---

## Features

- JWT Authentication
- Employee Management
- Inventory Management
- Product Management
- Supplier Management
- Brand & Category Management
- Dashboard Analytics
- Sales Reports
- PDF & Excel Export
- AI Assistant (Groq API)
- Dockerized Deployment

---

```bash
docker compose up
```

Services

Frontend → localhost:5173

Backend → localhost:8081
